# LibGui

_WARNING: This project was made to learn OpenGL, Git, CMake, some advanced C++ and making bigger codebases, 
so it's not intended for real usage._

LibGui is multiplatform GUI C++ library with text rendering/editing, image loading/rendering/editing and much more.

To learn more here is our [wiki](https://github.com/TechnologicalTurtle/LibGui/wiki).
